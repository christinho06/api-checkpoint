# API Checkpoint

A React application that fetches and displays a list of users from the [JSONPlaceholder API](https://jsonplaceholder.typicode.com/users) using Axios.

## Features
- Fetches user data using Axios inside useEffect hook
- Stores data in useState (listOfUser)
- Maps and displays all users with name, email, phone, website, company, and city
- Clean responsive card-based UI

## Setup
```bash
npm install
npm start
```
